# Namaste India – AI-Powered Tourism & Travel Planning Platform 

## Overview

Namastey India is a modern travel and tourism web application designed to help users explore the rich culture, heritage, history, and attractions of India. The platform combines an interactive user interface with AI-powered trip planning to provide a personalized travel experience.

The project aims to make discovering India easier by offering detailed destination information, smart travel assistance, and an intuitive browsing experience.

## Features
- Explore famous tourist destinations across India
- AI Travel Planner powered by Google Gemini API
- Intelligent chatbot for travel-related queries
- Interactive destination information
- Google Maps/Street View integration for location visualization
- Responsive design for desktop 
- Modern and user-friendly interface
- Dedicated pages for different Indian states and attractions
- Easy navigation with categorized travel destinations
- Search and explore travel locations effortlessly

## Tech Stack
### Frontend
- HTML
- CSS
- JavaScript

### Backend
- Node.js
- Express.js

### APIs & Services
- Google Gemini API (AI Travel Assistant)
- Google Maps / Street View

## AI Travel Assistant

The AI-powered travel assistant helps users by:

- Creating personalized travel itineraries
- Answering India-related travel questions
- Suggesting tourist attractions
- Recommending destinations based on user preferences
- Providing travel tips and planning assistance

## Current Status

This project is currently under development. At present, the **West Bengal** tourism page has been fully implemented as a prototype. Additional state pages and features will be added in future updates to expand the platform across India.
